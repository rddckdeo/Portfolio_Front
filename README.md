## 프로젝트 소개
- 프로젝트 기간 | 2025.11.27 ~ 2025.12.08
- Front-End : HTML, CSS, JavaScript 
    - Framework / Tools : node.js, React(CRA), Axios, VSCode
- Back-End : Java11
    - Framework / Tools : Spring Boot 2.7.18, Lombok, myBatis
- Database : MySQL
    - Tools : ERD Cloud, DBeaver
- ETC : GitHub, Postman, Ubuntu, Virtual Box
## 웹사이트 한 줄 요약 |<br/>
- 포트폴리오를 웹사이트로 구축 및 배포를 하기 위한 웹사이트입니다.

## 주요 기능
1. 프로필 사진 / 프로필 Info
  - 기본적인 연락처와 프로필사진을 보여주는 화면이며 각 사진별 hover 이벤트와 클릭에 따른 사진 변경 기능을 구현하였습니다.
![picture8](https://github.com/user-attachments/assets/9553c67b-91b6-4c40-8d7b-188a8a3227fc)
2. 프로젝트 Info
  - 지금까지 만들었던 프로젝트를 간단하게 소개해주는 화면으로 10초 주기로 자동으로 사진을 변경할 수 있게 구현하였습니다.
<img width="1708" height="800" alt="picture2" src="https://github.com/user-attachments/assets/1638b173-8f1b-431c-a925-0e3a21a579fe" />

3. 기술 스택
  - 현재 사용가능한 기술을 보여주기 위한 화면으로 DB에서 각 타입에(type : backend, frontend...) 따라 글자와 아이콘 ( isICon 아이콘 여부 )을 자연스럽게 배치되도록 구혀하였습니다.
4. Project Info
  - 구현한 프로젝틑를 좀 더 구체적으로 확인할 수 있도록 Card형태로 Grid 배치를 하였으며, 각 프로젝트별로 Click을 할 시 Popup창에 md 확장자 파일을 불러와 열 수 있도록 구현하였습니다.
<img width="1765" height="908" alt="picture3" src="https://github.com/user-attachments/assets/47d15c54-8252-45e5-be6f-dcadfaa732e9" />

5. 설정 파일 ( Setting )
  - Profile 부분의 리소스들을 관리하기 위한 설정파일로 프로필사진, 프로젝트, 기술 스택 등에 대해서 수정, 삭제, 사용 여부 등을 설정할 수 있습니다.
